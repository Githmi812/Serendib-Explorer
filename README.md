# Serendib-Explorer
🌿 Serendib Explorer
A travel website dedicated to showcasing the beauty of Sri Lanka across all 9 provinces. Built with HTML, CSS, and Bootstrap 5.
📄 Pages

Home — Hero landing with featured province destinations
Tour — Full card-based guide to places across every province with image carousels, locations, and map links
About — Our story, why choose us, and team values
Login / Register — Sign in and account creation with password strength checker

🛠️ Built With

HTML5, CSS3
Bootstrap 5.3
Google Fonts — Playfair Display, Cinzel, Lato
Font Awesome 6 icons

✨ Features

Fully responsive design (mobile & desktop)
Image carousels for each destination
Google Maps links for every location
Province-by-province browsing
Social login buttons (Google & Facebook)
Smooth scroll animations

Setup Instructions Back end
1. Import the Database

Open XAMPP Control Panel and start Apache and MySQL
Go to http://localhost/phpmyadmin
Click New, enter serendib_db and click Create
Click Import, select serendib_db.sql from the project folder and click Go

2. Run the Project

Copy the serendib folder into C:\xampp\htdocs\
Open your browser and go to http://localhost/Serendib-Explorer/index.html
